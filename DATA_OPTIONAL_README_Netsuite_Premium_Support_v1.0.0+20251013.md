# DATA OPTIONAL README Netsuite_Premium_Support v1.0.0+20251013

## Purpose
Supports troubleshooting and optimization by allowing uploads of NetSuite exports, reports, logs, and configuration snapshots.

## Accepted Formats
- CSV (NetSuite exports)
- Excel
- JSON/XML (integration logs)
- PDF (saved searches or workflows)

## Naming Convention
`DATA_TYPE_DESCRIPTION_vYYYYMMDD`

## Upload Instructions
Upload file into the project file section for the expert to access.

## Validation Step
Ask the expert: "Do you see the file X?"

## Maintenance Notes
Update with date-based versioning when re-uploading.
